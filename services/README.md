# Services

Long-running OpenAgentNet infrastructure nodes.

- `root-node`: trust anchor, authorization, bulletin, verified package publishing
- `registrar-node`: service agent registration and credentials
- `cdn-node`: local DID document and metadata distribution
- `discovery-node`: capability index and discovery API
